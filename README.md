## Development Environment 

### Node version

- v16.15.0

### npm version

- 8.5.5

### React version

- ^18.2.0


## Run project

In the project directory, you can run:

### `npm install`

To install all package using on projects. Then run:

### `npm start`

Then open [http://localhost:3000](http://localhost:3000) to view it in your browser.
